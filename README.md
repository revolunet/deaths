##### Install dependencies

```bash
yarn
```

##### Run development version

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

##### Run code linter

```bash
yarn lint
```

##### Build application

```bash
yarn build
```

##### Run built version

```bash
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
